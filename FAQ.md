## Назначение

«TeamSync Bot» — телеграм-бот + веб-панель для небольших команд (стартапов, учебных групп, фрилансеров). Бот помогает:

- генерировать идеи для проектов (с помощью AI),
- строить пошаговые планы разработки,
- назначать задачи и отслеживать прогресс,
- давать подсказки по коду / архитектуре.

## Целевая аудитория

- Начинающие IT-команды (2–5 человек)
- Студенты над курсовыми/командными проектами
- Solopreneurs, которым нужен «второй мозг»

## Ключевые функции

1. Генерация идей (по ключевым словам или голосом).
2. Построение плана (разбивка идеи на задачи с дедлайнами).
3. Совместная работа (назначение задач, статусы, комментарии).
4. Подсказки (примеры кода, best practices по запросу).
5. Интеграция с веб-дашбордом (React) — графики прогресса, список задач.
